API reference
Modifier class Effect
data-variant="landing" Display-scale title, wider content column
data-variant="case" Muted background, two-column with aside at ≥ 56rem
data-variant="vacancy" Two-column with aside; tighter content measure
data-variant="campaign" Muted background, two-column with aside
data-tone="inverse" Inverse surface and foreground tokens
data-variant="full-bleed" Span the full inline-size of the parent
data-align="center" Center-align title, copy, and actions
Class Purpose
.cui-page-intro-layout Grid wrapper for the content column and optional aside.
.cui-page-intro-content Content column — eyebrow, title, meta, lead copy, and actions.
.cui-eyebrow Kicker text rendered above the title.
.cui-page-intro-title Display heading for the page.
.cui-page-intro-meta Meta strip — typically a row of <span> labels (sector, year, stack).
.cui-page-intro-actions Primary and secondary CTA buttons.
.cui-page-intro-aside Optional sidebar card, shown to the right of the content at ≥ 56rem. Used by the case, vacancy, and campaign variants.
Variable Default

--cui-page-intro-title-size --cui-font-size-5
--cui-page-intro-bg · --cui-page-intro-color Surface and text overrides.
