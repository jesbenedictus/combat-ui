API reference
Class Purpose
.cui-hero Hero surface. Pair with a variant modifier (below).
.cui-hero-inner Centered, width-capped row wrapping the content and media.
.cui-hero-content Content column — eyebrow, title, lead copy, and actions.
.cui-eyebrow Small kicker text rendered above the title.
.cui-hero-title Display heading. Sized via --cui-hero-title-size.
.cui-hero-actions CTA button group — a flex row that wraps on narrow viewports.
.cui-hero-media Wraps the image or video for the split / overlay variants; the inner media covers the cell.
.cui-hero-bg Background-image layer for the background variant. Set the image with an inline --cui-hero-background-image custom property.
Modifier class Purpose
.cui-hero-split Shows the .cui-hero-media beside the copy.
.cui-hero-background Renders the .cui-hero-bg image behind the content.
.cui-hero-overlay Layers the content on top of a .cui-hero-media element with a darkening gradient.
.cui-hero-text Copy-only hero with tighter rhythm and a narrower container.
.cui-hero-media-start Places the media before the copy in the split layout.
.cui-hero-center Centers the content column (pair with .cui-hero-text).
.cui-hero-narrow · .cui-hero-default · .cui-hero-wide Caps the inner content's inline-size to a site container scale while the hero surface stays full-bleed. Defaults to wide (96rem); default aligns with the rest of the site content (72rem), narrow suits editorial copy (48rem). Drives --cui-hero-container-size.
