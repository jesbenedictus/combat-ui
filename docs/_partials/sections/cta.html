API reference
Class Purpose
.cui-cta-content Content stack — eyebrow, title, and supporting copy.
.cui-eyebrow Small kicker line above the title.
.cui-cta-title The call-to-action heading.
.cui-cta-actions One or more CTA buttons. Flex-wraps automatically.
Modifier class Purpose
.cui-cta-simple Centers the message and action for focused sections.
.cui-cta-split Places actions in a side column at ≥ 48rem.
.cui-cta-full-bleed Edge-to-edge muted band with section-scale padding and no card chrome.
.cui-cta-sticky Pins the CTA to the bottom of the viewport.
.cui-cta-inverse Inverse surface and text colors for high-contrast moments.
.cui-cta-center Center-aligns content and actions (implied by .cui-cta-simple).
Variable Default
--cui-cta-padding --cui-space-6
--cui-cta-gap --cui-space-5
--cui-cta-content-size 48rem
--cui-cta-title-size --cui-font-size-4
--cui-cta-bg · --cui-cta-color Surface and text overrides.
--cui-cta-inverse-bg · --cui-cta-inverse-color · --cui-cta-inverse-muted-color · --cui-cta-inverse-eyebrow-color Inverse-mode overrides (applied by .cui-cta-inverse).
