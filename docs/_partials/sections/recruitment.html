Recruitment

Vacancy cards, grids, lists, and a two-column detail layout for job boards. Each card is a semantic <article> whose title and apply action are real <a href> links, so listings stay crawlable and work without JS. The same card flexes from a minimal "title + read more" listing to an image-led row, and the detail layout pairs a narrative body with a sticky "Snel overzicht" summary, apply CTA, and contact block — enough to cover the range of formats real municipal recruitment sites publish.

Wat ga je doen?

Je loopt dagelijks je ronde door de Westwijk. Je bent zichtbaar, aanspreekbaar en signaleert wat er speelt — van leefbaarheid tot verkeersveiligheid.

  • Toezicht op leefbaarheid, verkeersveiligheid en overlast
  • Aanspreekpunt voor bewoners en ondernemers
  • Samenwerken met handhavers en politie in het kernteam

Wanneer hebben wij een match?

Je bent benaderbaar, luistert actief en blijft kalm onder druk. Een ronde lopen schrikt je niet af — je bent graag buiten en in beweging.

Wat wij jou bieden

Een salaris van minimaal € 2.495 en maximaal € 4.746 bruto per maand (schaal 5), ABP-pensioen, een Individueel Keuzebudget van 17,05% en een reiskostenvergoeding.

API reference
Class Purpose
.cui-surface.cui-vacancy-card One job opening. Compose .cui-surface for the card chrome (the card zeroes its padding so media bleeds; the body carries the inset). Default is a lean single column; add a first-child media element and data-orient="row" for an image-led card. data-variant="featured | compact | flat | borderless" swaps chrome; data-status="new | closing | closed" tones the corner pill. Carry data-team / data-location / data-contract for filtering.
.cui-vacancy-card-media Optional thumbnail (covers a bare first-child <img> / <a> too). Only shown in data-orient="row".
.cui-vacancy-card-eyebrow + .cui-eyebrow, -title + .cui-display, -tagline + .cui-lead, -excerpt, -status Body parts inside the card body (a .cui-stack): the team eyebrow composes .cui-eyebrow (keeps a retint knob + linked-label styling), the linked title composes .cui-display and the subtitle .cui-lead (both retuned to card scale via the type primitives' override vars), plus the excerpt and corner status pill. The action row is a .cui-cluster (bottom-aligned via a scoped rule).
.cui-meta data-variant="chips" The shared .cui-meta primitive carries the fact row used in the card and the detail header. data-variant="chips" renders each <li> as a rounded pill (hours, location, salary scale, contract); drop it for the inline ·-separated row.
.cui-grid Lay cards out with the .cui-grid primitive; set --cui-grid-min: 20rem for the roomier vacancy column. data-columns="2 | 3 | 4" pins fixed columns ≥48rem; data-gap="compact | spacious" adjusts spacing.
.cui-stack Single-column overview is the .cui-stack primitive (set --cui-stack-gap for compact / spacious rhythm). data-variant="stacked" on the stack collapses vacancy-card chrome to hairline separators for a dense index.
.cui-vacancy-detail Main + aside grid for a detail page; container-query driven (flips at 52rem inside a .cui-page ancestor), collapses to one column below it. data-aside="start" moves the aside before the body.
.cui-vacancy-header + .cui-stack, .cui-vacancy-main, .cui-vacancy-section Full-width title band (composes .cui-stack), narrative body column (capped measure), and a titled body section that renders cleanly whatever heading a posting uses.
.cui-vacancy-aside + .cui-surface.cui-stack, .cui-vacancy-summary, .cui-vacancy-apply / .cui-vacancy-contact + .cui-stack Sticky aside rail — composes .cui-surface (muted background, no shadow) and .cui-stack — holding a quick-facts <dl> ("Snel overzicht"), a full-width apply CTA with helper note, and a contact-person block (both compose .cui-stack).
