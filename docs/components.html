Components

Interactive primitives, grouped by what they do. Each custom element manages behavior and ARIA while links, buttons, and content stay in light DOM for styling and SEO. For full-page building blocks see Sections; for form-specific elements see Forms.

Navigation

cui-navbar with sticky positioning, responsive collapse, dropdowns, and nested menus.

Atoms

Buttons, tab panels, code blocks, and the cui-tree element — small primitives that compose into larger surfaces.

Motion

cui-scroll-stage publishes focus and offset as CSS custom properties; cui-reveal layers entry animations on top. Reduced-motion aware.

Overlays

cui-modal over native <dialog>, cui-disclosure over <details>, and the toast() notifications API.
